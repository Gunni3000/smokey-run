# Smokey Run!

A portrait endless-runner grey-box prototype for mobile browsers.

## Run locally

```bash
npm start
```

Open `http://127.0.0.1:4173`.

## Controls

- Tap **Jump** or press `Space` / `ArrowUp`.
- Jump over boxes and shoes.
- Collect tuna and chicken to gain distance from the carrier.

The best survival time is stored locally in the browser.
